# ccat

`ccat` is the colorizing `cat`. It works similar to `cat` but displays content with syntax highlighting.

# Installation

## From source

```
$ go install github.com/jingweno/ccat
```

# Examples

```
$ ccat FILE1 FILE2 ...
$ ccat # read from standard input
```
